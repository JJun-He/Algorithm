### 문제 설명

정수 A, B, C가 주어졌을 때, 다음을 만족하는 정수 x, y를 구하는 프로그램을 작성하시오.

Ax + By = C

만약 정수해가 존재하지 않으면 -1을 출력한다.

### 입력

한 줄에 정수 A, B, C (-10^9 ≤ A, B, C ≤ 10^9)가 주어진다.

### 출력

정수해 x, y를 공백으로 구분해 출력한다.

만약 정수해가 존재하지 않으면 -1을 출력한다.

해가 여러 개 존재할 경우 아무 해나 출력한다.
